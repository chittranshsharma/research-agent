## 🚀 Getting Started (Run the Future Locally)

Prepare your machine for the most advanced application it has ever run.

### 1. Clone the Repository
```bash
git clone https://github.com/chittranshsharma/research-agent.git
cd research-agent
```

### 2. Ignite the Backend (FastAPI)
```bash
cd backend
python -m venv .venv
# On Windows:
.\.venv\Scripts\activate
# On Mac/Linux:
# source .venv/bin/activate
pip install -r requirements.txt

# Create your .env file and add your API keys (e.g. OpenAI, Tavily)
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```
*The AI brain is now awake on `http://localhost:8000`.*

### 3. Launch the Frontend (Next.js)
```bash
cd frontend
npm install
npm run dev
```
*The portal to the future is open at `http://localhost:3000`.*
